Changelog

This document tracks all notable changes to the Pestector Plant Disease Classifier application from version 1.0.0 to 6.1.2.

v6.1.2 Released on February 24, 2025

New Features

  • A background scheduler is used to clean up old files periodically.
  • Added enhanced PDF report generation with professional styling and multi-page layout
  • Integrated watermark and confidentiality notices in generated reports
  • Added support for direct download of reports via API endpoint

Improvements

  • Optimized image preprocessing pipeline for faster analysis
  • Enhanced metadata extraction from uploaded images
  • Improved error handling and logging for better diagnostics

Bug Fixes

  • Fixed image resizing issues with non-standard aspect ratios
  • Resolved PDF generation failures on certain operating systems
  • Fixed incorrect confidence level display in the API response
v5.3.0 Released on February 18, 2025

New Features

  • Integrated Gemini API for dynamic treatment recommendations
  • Added disease pattern recognition system to identify common symptoms
  • Implemented API key authentication for secure access

Improvements

  • Enhanced model prediction speed by 35%
  • Added detailed confidence level classification (High/Medium/Low)
  • Improved image metadata extraction and classification

Bug Fixes

  • Fixed memory leak in the model prediction pipeline
  • Resolved CORS issues with API endpoints
  • Fixed classification errors for specific plant species
v4.2.1 Released on February 14, 2025

New Features

  • Implemented top-3 predictions with confidence scores
  • Added disease severity classification system
  • Created new scan page with improved UI/UX for mobile devices

Improvements

  • Updated VGG model with improved accuracy on rare plant diseases
  • Enhanced logging system for better troubleshooting
  • Optimized image upload process for larger files

Bug Fixes

  • Fixed incorrect treatment recommendations for certain plant species
  • Resolved file upload issues on Safari browsers
  • Fixed inconsistent confidence score calculations
v3.0.0 Released on January 25, 2025

New Features

  • Complete redesign with FastAPI framework for improved performance
  • Added automatic categorization of plant diseases by type
  • Implemented health check endpoint for system monitoring
  • Added CORS middleware for cross-origin requests

Improvements

  • Upgraded to TensorFlow 2.x for model deployment
  • Enhanced model singleton pattern for better resource management
  • Improved error handling with detailed error messages

Breaking Changes

  • Restructured API response format for compatibility with mobile apps
  • Changed authentication method from session-based to API key
  • Moved static files to dedicated directory structure
v2.1.0 Released on January 16, 2025

New Features

  • Added support for 10 new plant species and diseases
  • Implemented basic PDF report generation
  • Added categories endpoint to list available plant species

Improvements

  • Optimized image preprocessing for better prediction accuracy
  • Enhanced model loading with error handling
  • Improved static file serving performance

Bug Fixes

  • Fixed model loading failures on startup
  • Resolved EXIF data extraction issues
  • Fixed confidence calculation for edge cases
v1.0.0 Released on January 2, 2025

Initial Release

  • Basic plant disease classification with VGG model
  • Support for 38 plant disease classes
  • Simple web interface for image upload
  • JSON response with plant and disease identification
  • Basic confidence score calculation
  • Image upload and storage functionality
